# 🏦 Bank Management System

A console-based **Bank Management System** developed in **C++** using **Object-Oriented Programming (OOP)** principles and **Binary File Handling**. The application enables users to create bank accounts, perform transactions, and manage account information through a menu-driven interface.

---

## 🚀 Features

- Create a new bank account
- Deposit money into an account
- Withdraw money from an account
- Display all customer accounts
- Persistent data storage using binary files
- Menu-driven console interface

---

## 🛠️ Tech Stack

- **Language:** C++
- **Concepts:** Object-Oriented Programming (OOP)
- **Storage:** Binary File Handling
- **Compiler:** GCC / g++

---

## 📚 OOP Concepts Used

- Classes & Objects
- Encapsulation
- Data Abstraction
- Member Functions

## ▶️ How to Run

### Clone the repository

```bash
git clone https://github.com/Shashwat-pandey21/Bank-Management-System.git
```

### Navigate to the project folder

```bash
cd Bank-Management-System
```

### Compile the program

```bash
g++ main.cpp -o bank
```

### Run the executable

#### Windows

```bash
bank.exe
```

#### Linux / macOS

```bash
./bank
```
